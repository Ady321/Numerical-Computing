Truncation Error


    Its a difference between truncated value and actual value. The truncated quantity is represeted by numerical with
fixed  number of allowed digit with any access digit is removed.

Round Off Error


    Its a difference between rounded of numeric value and actual value. A rounded quantity is represented by numeric with 
fixed number of allowed digits with last digit set to value that produces smallest diference between rounded quantity 
and actual quantity.

Absolute Error


    It is magnitude of difference between exact value and approximate value.

Relative Error


    It is obtain by didviding the absolute error by exact value.

Percentage Error


    When relative error is expressed in terms of percentage it is called percentage error.
