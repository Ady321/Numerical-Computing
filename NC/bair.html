Steps For Solving Bairstow Method


  1. Find the the value of a0,a1,a2 upto an by comparing equation P(x)=a0xn + a1xn-1 + ----- + an = 0
with equation given in the problem also find the value of n i.e, the highest degree of equation.

    2. 

  2. Find the value of interval b1,b2,b3 upto bn by using formula
                 bk = ak - p*bk-1 - q*bk-2 with b-1 = 0

    3. 

  3. Find the value of interval c1,c2,c3 upto cn by using formula
                 ck = bk - p*ck-1 - q*ck-2 with c-1 = 0

    4. 

  4. Find the value of Δp and Δq by using formula
                Δp = -[bn*cn-3 - bn-1*cn-2]
                                               
                     [c2n-2 - cn-3[cn-1 - bn-1]]
                
                Δq = -[bn-1(cn-1 - bn-1) - bn*cn-2]
                                                 
                       [c2n-2 - cn-3[cn-1 - bn-1]]

    5. 

  5. Find the value of p1 and q1 by using formula
                p1 = p0 + Δp
                q1 = q0 + Δq

    6. 

  6. Continue the steps 2-5 for number of iteration given in the problem.

    7. 

  7. Obtain the quadratic factor by using the equation x2 + px + q and by putting the final values of p and q.
