Steps For Solving Gauss Elimination Method


  1. Suppose we consider the system of equation with n unknowns
   a11*x1 + a12*x2 + ----- + a1n*xn = b1
   a21*x1 + a22*x2 + ----- + a2n*xn = b2 
   		  ¦
                  ¦  
   an1*x1 + an2*x2 + ----- + ann*xn = bn

    2. 

  2. Then unknowns are eliminated to obtain an upper traingular matrix to eliminate x1 from second equation we must 
multiply 1st equation by (-a21)/a11 and add to the second equation is 
(a22 - a12*a21)*x2 + (a23 - a13*a21)*x3 + ---- = b2 - b1*a21
           a11                 a11                      a11
we can return as a22'*x2 + a23'*x3 + --- + a2n'*xn = b'2

    3. 

  3. Similarly Elimianate the x1 from third equation is a32'*x2 + a33'*x3 + --- + a3n'*xn = b'3

    4. 

  4. After elimnating x1 from the system of equation is
      a11*x1 + a12*x2 + ----- + a1n*xn = b1
              a22'*x2 + ----- + a2n'*xn = b'2 
   		     ¦
                     ¦  
              an2'*x2 + ----- + ann'*xn = b'n

    5. 

  5. The first equation of the system is called the pivot equation and a11 is called as pivot element. Then we can
eliminate x2 and then x3 and so on.
    6. 

  6. We form the uppertraingular matrix
      a11*x1 + a12*x2 + ----- + a1n*xn = b1
              a22'*x2 + ----- + a2n'*xn = b'2 
   		              ¦
                              ¦  
                              ann(n-1)*xn = bn(n-1)
 Where ann(n-1) indicates that the element ann has changed (n-1) times
